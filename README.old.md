# What is this？

moe-container-manager is a port of Linux cli environment that cannot run docker and kvm, based on termux-container.

# Quick start

### WARNING:      

- It is not finished and it doesn't work right now.

### for Debian/Ubuntu
```
cd moe-container-manager
sudo apt build-dep .
make
make install
```

### for Arch Linux
```
sudo pacman -Syy p7zip golang unzip zip git wget curl nano proot axel pv gawk gettext git gcc g++ make libcap libseccomp go cmake libspdlog ninja-build
cd moe-container-manager
make
make install
```

### for Other Linux
install libcap,gcc,libspdlog,golang,libseccomp,cmake,ninja-build and make。
```
cd moe-container-manager
make
make install
```
### build deb package
```
cd moe-container-manager
sudo apt build-dep .
dpkg-buildpackage -b -us -uc
```

### build archlinux pkgs(Experimental features)
```
cd moe-container-manager
cd tools/pkgs
python3 makepkg.py
```
